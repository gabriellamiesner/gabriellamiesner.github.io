Neurosight is a project built over the course of Tech Together New York. The web application aims to improve the online learning experience. I contributed auto updating graphs for the website to mimic what assignment analytics would look like. I used python and matplotlib create the data visualizations, and created a python script to generate dummy student score data as we did not have any on hand. Neurosight won best survival hack at TechTogether New York.

Renew was created as a final project for the 2020 Web Development Kode with Klossy Summer Program. The application aims to allow the user to interact with the WashU Climate Report on Environmental Racism in Saint Louis in a more interactive and accessible way. THe website includes the information of the report, as well as information on representatives to contact, further reading, and local organizations fighting for climate justice.

Climate Clock was created over the course of TechTogether Boston. The web application aims to educate the user about the history of the environmental justice movement in an interactive timeline format. When the user hovers over an item, the item expands to include more information about the event in environmental justice history.
